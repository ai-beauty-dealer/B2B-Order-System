# 取り込みモード（LINE文面 → カート）セットアップ・運用手順

## できること

MASTERアカウントでサロンに入室すると、ヘッダーに「📥 取り込み」ボタンが出る。
サロンから届いたLINEの発注文面を貼り付けて「解析する」を押すと、AIが

1. そのサロンの発注履歴（過去62日）＋お気に入り＋使用メーカーのカラー剤色番 を候補にして
2. 文面の各行を商品コード・数量にマッチングし
3. 確信度つきのプレビュー（緑=ほぼ確実／黄=推定／橙=要確認／赤=未マッチ）を表示する

数量を確認・修正してチェックした行だけカートに入る。**発注確定は今まで通り「発注する」ボタンから**（自動発注はしない）。

- 一般サロン・GROUPアカウントにはボタン自体が表示されない
- シートへの書き込みは一切しない読み取り専用アクション（二重発注ガード等の既存ロジックに影響なし）

## 初回セットアップ（2ステップ）

### 1. APIキーの設定

1. [Anthropic Console](https://console.anthropic.com/) でAPIキーを発行する
2. Apps Script エディタ →「プロジェクトの設定」→「スクリプト プロパティ」
3. 追加：キー `ANTHROPIC_API_KEY`／値 `sk-ant-...`（発行したキー）

### 2. Apps Script へのファイル追加とデプロイ

1. B2B発注サイトの Apps Script プロジェクトを開く
2. 「ファイル」横の `+` →「スクリプト」→ 名前 `order_import` で作成
3. ローカルの `backend/order_import.gs` の全内容を貼り付けて保存
4. `code.gs` を開き、ローカルの `backend/code.gs` を全量コピーして貼り替え（`parse_order` のルーティング1行が追加されている）
5. エディタ内検索で以下があることを確認：
   - `function handleParseOrder`（order_import 側）
   - `action === 'parse_order'`（code.gs の doPost 内）
6. 「デプロイを管理」から Webアプリの新バージョンをデプロイ
7. フロント（index.html / app.js / style.css / sw.js＝v2.26.0）を GitHub Pages に push

## 動作確認

1. MASTERアカウントでログイン → サロンを選んで入室
2. ヘッダーに「📥 取り込み」が出ることを確認（一般サロンIDでログインし直すと出ないことも確認）
3. 実際のLINE発注文面を貼り付けて「解析する」（30秒前後かかる）
4. プレビューの商品・数量が合っているか確認 → チェックした行がカートに入る
5. カートから通常どおり発注確定できる

## 運用ルール

- **数量は必ず目視確認**。特に黄・橙の行（推定・要確認）は元の記載と見比べる
- 赤の「特定できなかった行」は、検索か「＋特注・その他」で手動追加する（テラカラー等のマスタ外取寄せ品はここに出るのが正常）
- AIが別名を読み間違えたら、`docs/取り込みモード_別名辞書.md` に正体を追記し、`order_import.gs` の `PARSE_ALIAS_HINTS` にも同じ行を足して再デプロイする
- コスト目安：1回の解析で**1円以下**（Claude Haiku運用時）。月100回でも数十円
- **精度がイマイチなとき**：`order_import.gs` 冒頭の `PARSE_MODEL` を `'claude-opus-4-8'` に変更＋payloadに `thinking: { type: 'adaptive' }` を追加して再デプロイ（1回数円〜10円になるが精度最優先の構成）。手順はファイル内コメント参照

## トラブルシューティング

| 症状 | 原因と対処 |
|---|---|
| 「ANTHROPIC_API_KEY が未設定です」 | スクリプト プロパティにキーを追加してデプロイし直す |
| 「AI解析に失敗しました（HTTP 401）」 | APIキーが無効。Consoleで再発行して差し替え |
| 「AI解析に失敗しました（HTTP 429）」 | レート制限。1分待って再実行（自動で1回リトライ済み） |
| ボタンが出ない | MASTERでログインしているか確認。PWAは一度閉じて開き直す（v2.26.0が配信されているか） |
| 解析が遅い・タイムアウト | 文面が長すぎる場合は分割して貼る（上限4000字） |
